# 🚀 LeadZone — Modern Lead Management

LeadZone is a clean, focused CRM built for managing and tracking sales leads efficiently.  
It keeps the workflow simple: capture leads, update their status, schedule follow-ups, and stay organized.

🌐 **Live App:** https://lead-zone.vercel.app  

---

## ✨ What This Project Does

LeadZone is designed around one core idea:  
**make lead tracking straightforward without unnecessary complexity.**

You can:

- ➕ Add new leads with contact details and budget
- ✏️ Edit lead information at any time
- 🗑️ Remove leads you no longer need
- 🔎 Search leads instantly
- 🎯 Filter by status
- 📅 Schedule follow-ups with date & time
- 🔒 Access everything through secure authentication

---

## 🧠 Key Features

### 🔐 Authentication
- Email/password login
- Protected dashboard access
- User-specific data isolation

### 📊 Lead Dashboard
- Clean UI layout
- Real-time data updates
- Status-based organization
- Date sorting (newest / oldest)

### 🗂️ Smart Filtering
- Search by name, email, or number
- Filter by lead status
- Combined filtering + sorting logic

### ⏰ Follow-Up Scheduling
- Date & time picker integration
- AM/PM formatting
- No past-date selection

---

## 🏗️ Tech Stack

- **React (Vite)** — Frontend framework
- **Supabase** — Authentication & database
- **Tailwind CSS** — UI styling
- **Flatpickr** — Date/time handling
- **Vercel** — Deployment platform

This project demonstrates:
